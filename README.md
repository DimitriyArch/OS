## 1. Git Overview
Git — це система керування версіями, що дозволяє зберігати історію змін коду, працювати над проектом кільком розробникам одночасно та повертатися до попередніх версій у разі помилок.

**Основні команди:**
- `git add` - підготовка файлів.
- `git commit` - збереження версії.
- `git push` - відправка у хмару.

## 2. What is a Commit?
Комміт — це логічна одиниця збереження змін. Він дозволяє бачити різницю (diff) між поточним та попереднім станом файлів, що робить процес розробки прозорим.
