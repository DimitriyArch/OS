
Операційні Системи
================

Ядро Linux — це основа будь-якої операційної системи Linux. Воно керує апаратним забезпеченням,
системними ресурсами та надає базові служби для всього іншого програмного забезпечення.

[ [English](README.md)  |   Українська ]

Основна Документація
-----------------------

Це наш навчальний/студентський репозиторій для вивчення операційних систем, переважно на базі GNU/Linux і частково Windows NT.

Посилання на необхідні ресурси:

* [Google Classroom](https://classroom.google.com)
* [Cisco Network Academy](https://www.netacad.com/dashboard)
* [Network Development Group](https://portal.netdevgroup.com/learn)
